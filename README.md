# SpendLens

A privacy-first Android app that reads your bank/UPI/card **transaction SMS**, parses them with a
**self-learning pattern engine**, detects **duplicate transactions**, and presents a beautiful
**Material 3 expense-analysis UI**.

> **Everything runs on-device.** Raw SMS, account numbers, and amounts never leave the phone.
> The database is encrypted (SQLCipher + Android Keystore) and the on-device build ships with
> **no Internet permission**.

## Key decisions (locked)

- **Native Kotlin + Jetpack Compose** (Material 3)
- **Fully on-device**, encrypted local DB (Room + SQLCipher)
- **Pluggable AI** `PatternGenerator` interface, **stubbed** now with a no-network heuristic fallback
- **Generic / multi-region** built-in pattern set (not tied to one bank/country)

## Build & run (once Android Studio is installed)

**Requirements:** Android Studio (Koala or newer), JDK 17+ (you have JDK 21 ✓), an Android 8.0+
device or emulator. The Android SDK is installed by Android Studio.

1. **Open** the `SpendLens` folder in Android Studio (*File → Open*). Let it sync Gradle — it will
   download the Android Gradle Plugin and SDK packages on first sync.
2. The project uses a Gradle **version catalog** and **wrapper** (`gradle-8.9`); no global Gradle
   install needed.
3. **Run** the `app` configuration on a device/emulator (▶).
4. Grant the SMS permission when prompted (see the privacy note below).

CLI alternative (after the SDK exists and `local.properties` points at it):
```powershell
# from the SpendLens folder
.\gradlew.bat assembleDebug      # build the debug APK
.\gradlew.bat testDebugUnitTest  # run JVM unit tests (parser/dedup)
```
> Note: the Gradle wrapper `.jar`/scripts are generated by Android Studio (or `gradle wrapper`)
> on first use. If `gradlew.bat` is missing, run *File → Sync Project with Gradle Files* once.

## Privacy & permissions

- **`READ_SMS` / `RECEIVE_SMS`** — to read existing transaction SMS and parse new ones. Requested at
  runtime with an explanation.
- **`POST_NOTIFICATIONS`** (Android 13+) — to notify when new transactions are detected.
- **No `INTERNET` permission** in this build — verifiably on-device.
- Database encrypted at rest; cloud backup / device transfer disabled.
- Distributing an SMS-reading app on Google Play requires the SMS-permission policy /
  Default-SMS-handler review; for personal use you can sideload the debug APK.
